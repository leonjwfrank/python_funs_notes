# 矩阵定义
    AB 称为 A 左乘 B，或B右乘A
     
# 矩阵乘性质

    k是数，A，B，C是矩阵
    1,(AB)C = A(BC)
    2,A(B+C) = AB + AC
        (A+B)C = AC + BC
    3,kAB=(kA)B = A(kB)
    
    
    *幂
    方阵A
        A**n = A*A*A...(n个)
        A和B为同阶方阵，则AB = BA
        对于任意正整数k，(AB)**k = A**kB**k
        
        
    (AB)**2 != A**2B**2   不一定
    
    *结合率
    
    例:
        A=BC 求 A**100 , 其中C = [1,-1]  B = [1
                                 2] 
                                 
       CB = -1                         
       A = [1
            2]  * [1,-1] **100
         = (BC)(BC)...   #100次
         = B(CB)(CB)(CB)...(CB)(CB)C = B(-1)**99C
         = B(-1)C = -(BC) = -A 
         =[-1  1
           -2  2]
