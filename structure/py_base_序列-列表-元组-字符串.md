        alist[2]=50     # 改变alist序号为2的元素的值，这个时候A的 所以元素都改变了
        
        print(A)
        [[1,2,50,4],[1,2,50,4],[1,2,50,4]]

    h3. 拷贝
        浅拷贝
            新建一个对象
               对象类型同原对象
               内容为源对象内容的引用
            默认类型拷贝
               [:]  完全切片操作
               list(),dict() 工厂函数
               copy.copy()  copy模块
            区分
                字符串 
                    新建对象 +显示拷贝
                列表元素
                    新建对象 是复制引用
            如list拷贝
            不会递归地复制内部对象。它仅制作最外层列表的副本，同时仍引用前一个变量的内层列表，因此，当您对内层列表进行突变时，更改将同时反映在原始列表和浅表副本中。
        
            >>> a = [[1, 2, 3], [4, 5, 6]]
            >>> b = list(a)
            >>> a
                [[1, 2, 3], [4, 5, 6]]
            >>> b
                [[1, 2, 3], [4, 5, 6]]
            >>> a[0][1] = 10
            >>> a
                [[1, 10, 3], [4, 5, 6]]
            >>> b   # b changes too -> Not a deepcopy.
                [[1, 10, 3], [4, 5, 6]]

        深拷贝
            copy.deepcopy()  新建对象+全选引用
            >>> import copy
            >>> b = copy.deepcopy(a)
            >>> a
                [[1, 10, 3], [4, 5, 6]]
            >>> b
                [[1, 10, 3], [4, 5, 6]]
            >>> a[0][1] = 9
            >>> a
                [[1, 9, 3], [4, 5, 6]]
            >>> b    # b doesn't change -> Deep Copy
                [[1, 10, 3], [4, 5, 6]]   
                             
        h4. 非容器类型没有拷贝
                数字,字符串,其他'原子'类型对象浅拷贝用[完全切片操作]完成
            元组变量只包括源自类型对象, 不会进行深拷贝,只能得到浅拷贝
