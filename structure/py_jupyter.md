# jupyter
    h2.   安装
            pip3  install   jupyter
        
    h2.    启动
            进入相应的目录，启动终端，jupyter   notebook
           
    h3.    选项菜单
            notebook 的名称
            主工具栏，提供了保存、导出、重载 notebook，以及重启内核等选项
            快捷键
            notebook 主要区域，包含了 notebook 的内容编辑区

    h4.     访问地址
          http://localhost:8888/
